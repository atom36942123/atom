Atom is a soon to be open-source backend that can be used to build applications faster. 
Atom has a complete set of pre-built modules that can be used to assemble production-ready applications in weeks instead of months. 

Tech Stack

1. Backend - Python/FastAPI
2. Primary Database - Postgres
3. Secondary Database - MongoDB
4. Retool - for admin panel
5. Redis layer included for caching
6. Deployment on AWS configured
7. Deployment on azure,gcp etc possible
8. Staging/Production/others env
9. Api Collection/Runner on postman
10. Scalability in terms of database schema
